Weather Monitoring System

   This Weather Monitoring System Project is a simple IoT project used to collect Weather Data using deployed IoT Weather Nodes which are configured in Star Topology. The deployed nodes sends data to a flask based API hosted on Heroku Platform that uploads data to Postgres DB. The stats can be monitored via the React Based Weather Dashboard Application. Each Node consists of 3 main sensors:

BMP-280

BMP-280 is a low cost sensor that monitors Temperature, Pressure and Altitude with high precision and is a successor to BMP-180. BMP-280 utilizes I2C/IIC protocol for communication with the micro-controller.

DHT-22

DHT-22 is a low power Temperature and Humidity Sensor with relatively high accuracy. This is used to monitor the Humidity and also for cross referencing with the BMP-180 sensor Temperature readings. It has a single data pin for getting data from sensor.

ML8511 (GYML8511 breakout)

ML8511 is a low cost high accuracy UV intensity sensor. This is used to measure the UV index of the surroundings. This uses the data and the voltage readings from sensor to calculate the UV intensity.

   All the data collected via these sensors is sent to a Flask based API which stores the data in a postgres database. The data is sent via HTTPS protocol by an ESP-12E module. ESP-12E is a low cost WiFi module that can connect to any access point on a 2.5G band and supports IEEE 802.11 b/g/n protocol. This module can act as an Access Point or a Station or both. The data collected can be accessed via web based React Dashboard or the React Native Mobile App. These dashboards can also show if a device is down and needs repairs or if any kind of maintainance is required.

RTC Clock (DS3231)

RTC Clock is based on DS3231 RTC chip. This chip provides accurate date and time once finishing first time program setup of clock. This chip has a 3.3V Lithium Button Battery for auxilary power for preventing time config loss

128x64 I2C Oled

This is a small Oled display which can be used for displaying the data or any ASCII graphics. In this case this displays the recorded data from sensors. This specific display is of 128px in length and 64px in breadth.

LM7805 Voltage Regulator

LM7805 is a 3 terminal Linear Voltage Regulator which takes the input voltage from the LiPo battery and converts it into 5V DC. It can output a max current of 1.5A. Input voltage ranges from 7V to 25V.

ESP12E Wireless Module

It is a power efficient wifi module with a lot of memory and GPIO options. It supports various protocols like I2C, SPI, UART, etc. It supports IEEE 802.11 b/g/n protocols. It can also support sleep on idle and works on 2.5G band.

STM32F103C8T6 (Blue Pill)

It is a small low cost high performance microcontroller by STMicroelectronics. It supports 3.3V input voltage and accepts 5V on only a select few pins. It comes with 64KB flash, 20KB SRAM and has a max clock speed of 72MHz. It supports I2C,SPI,UART,CAN and USB communication protocols.
