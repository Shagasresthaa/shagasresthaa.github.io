Weather Monitoring System

   This Weather Monitoring System Project is a simple IoT project used to collect Weather Data using deployed IoT Weather Nodes. The deployed nodes sends data to a flask based API that uploads data to Postgres DB. The stats can be monitored via the React Based Weather Dashboard Application. Each Node consists of 3 main sensors:

BMP-180 (GY-68)

BMP-180 is a low cost sensor that monitors Temperature, Pressure and Altitude with high precision. BMP-180 utilizes I2C/IIC protocol for communication with the micro-controller.

DHT-22

DHT-22 is a low power Temperature and Humidity Sensor with relatively high accuracy. This is used to monitor the Humidity and also for cross referencing with the BMP-180 sensor Temperature readings. It has a single data pin for getting data from sensor.

ML8511 (GYML8511 breakout)

ML8511 is a low cost high accuracy UV intensity sensor. This is used to measure the UV index of the surroundings. This uses the data and the voltage readings from sensor to calculate the UV intensity.

   All the data collected via these sensors is sent to a Flask based API which stores the data in a postgres database. The data is sent via HTTPS protocol by an ESP-8266 module. ESP-8266 is a low cost WiFi module that can connect to any access point on a 2.5G band. This module can act as an Access Point or a Station or both. The data generated is also logged in an on-board micro-SD module so that it can act as backup at any time. The data collected can be accessed via web based React Dashboard or the React Native Mobile App. These dashboards can also show if a device is down and needs repairs or if any kind of maintainance is required.

RTC Clock (DS3231)

RTC Clock is based on DS3231 RTC chip. This chip provides accurate date and time once finishing first time program setup of clock. This chip has a 3.3V Lithium Button Battery for auxilary power for preventing time config loss

128x64 I2C Oled

This is a small Oled display which can be used for displaying the data or any ASCII graphics. In this case this displays the recorded data from sensors. This specific display is of 128px in length and 64px in breadth.

AMS1117 Voltage Regulator

ML8511 is a low cost high accuracy UV intensity sensor. This is used to measure the UV index of the surroundings. This uses the data and the voltage readings from sensor to calculate the UV intensity.

ESP8266-01 WiFi Module

It is a low power wifi module with esp8266 chip at its core. It has 2 GPIO pins of which one can be connected to ground for programming the ESP module.

Micro SD module

This is a low cost module for attaching a micro SD card for data logging. Supports upto an 8GB SD Card. Can be used easily by standard I/O Libraries.

Atmega 328P PU

It is a small low cost low power CMOS 8-bit micro-controller based on AVR optimised RISC Architecture. It has 32x8 General Purpose Working Registers and an inbuilt ADC (Analog to Digital Converter). It supports UART, I2C/IIC and SPI communication. It has 1KB EEPROM, 2KB of SRAM, 32KB of flash of which 0.5KB is used by bootloader and upto 16MHz clock speed.

   All this data is controlled via two ATMEGA 328P micro-controllers. These are small 8-bit AVR optimised RISC based micro-controllers. Of the two microcontrollers, Primary micro-controller handles sensor data-monitoring and displaying it on the onboard oled display and sending it to the secondary microcontroller while the secondary microcontroller logs the data to the onboard SD card module and sends data to ESP8266 for data transmission. Both the micro-controllers communicate via I2C bus in which one acts as Primary Chip while other acts as Secondary Chip.
